# Architecture Decision Records (ADRs)

One file per decision: `NNNN-<slug>.md` (zero-padded, incrementing). Use the template at
`.claude/skills/feature-architecture/assets/ADR.template.md`. Each ADR carries its context,
the decision, the Mermaid diagrams, consequences, and rejected alternatives.

Don't edit an Accepted ADR to change the decision — add a new ADR that supersedes it.
